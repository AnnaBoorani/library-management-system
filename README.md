#  Library Management System

A web-based Library Management System built using **React.js, JavaScript, HTML, and CSS**.  
The application allows users to manage books with add, edit, delete, and search functionality and stores data using **LocalStorage**.

##  Features
- Add, Edit, Delete Books
- Search books by Title, Author, or Genre
- Library details management (Name, Address, Contact)
- Persistent data using LocalStorage
- Clean and responsive UI

##  Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- LocalStorage
